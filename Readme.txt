Steps:

1. Install niftyreg before run these matlab scripts.

	http://cmictig.cs.ucl.ac.uk/wiki/index.php/NiftyReg

2. replace the working path including your own data

To build model, follow the following steps:

1. get_wk_avg_flow.mat is used to generate stationary velocity fields.

2. AM.m is used to create average model

3. RIM.m is used to create refined individual model.

The spot_anatomy_orig_new.m is used to calculate the WEPL.
